# Migrate Website from cPanel to aaPanel

## 📌 Overview
This guide explains the migration of a website from cPanel to aaPanel, including files and database.

---

## ⚙️ Key Steps

- Backup website files and database from cPanel
- Transfer files to new server
- Import database into aaPanel
- Configure website and verify functionality

---

## 🔐 Migration Considerations

- File permissions
- Database configuration
- DNS updates if required

---

## ⚠️ Common Issues

- Database connection errors
- Missing files
- DNS propagation delay
