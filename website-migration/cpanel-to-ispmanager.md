# Migrate Website from cPanel to ISPmanager

## 📌 Overview
This guide covers the migration of a website from cPanel to ISPmanager.

---

## ⚙️ Key Steps

- Backup website and database from cPanel
- Upload files to ISPmanager server
- Import database and configure access
- Setup domain and test website

---

## 🔐 Migration Considerations

- File structure compatibility
- Database credentials
- DNS configuration

---

## ⚠️ Common Issues

- Permission errors
- Database import issues
- Service misconfiguration
