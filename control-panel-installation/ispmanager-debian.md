# Install ISPmanager on Debian

## 📌 Overview
This guide outlines the installation of the ISPmanager hosting control panel on a Debian-based server.

---

## ⚙️ Key Steps

- Prepare Debian server with required resources
- Download and run ISPmanager installation script
- Complete installation and access control panel
- Perform initial configuration

---

## 🖥️ System Requirements

- Debian OS (10 / 11 / 12)
- Root or sudo access
- Stable internet connection

---

## 🔐 Initial Setup

- Access panel via server IP and default port
- Complete setup wizard
- Configure hostname and user access

---

## ⚠️ Common Considerations

- Ensure required ports are open
- Verify network connectivity
- Check system compatibility before installation
